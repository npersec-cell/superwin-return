# Project Conventions & Preferences

- **Frontend Theme**: Binance-inspired dark trading platform UI. Core colors: dark background `#0d1013`, bright yellow `#ffe100` for CTAs and main accents, and flat dark cards `#20252b`.
- **Language Convention**:
  - **Public Frontend**: Must be 100% English. No Thai on public pages or user modals.
  - **Admin Backend (`/admin`)**: Must be 100% Thai. Labeled buttons, forms, status, and error messages are in Thai.
- **Layout Rhythm**: Narrower app container (max-width `820px`), snug spacing, auto page height, 34px-40px tight controls, and small-to-medium radii (8-12px) with distinct non-clipped elements.
- **Data & Facts**: Esport prediction pool focus. Avoid showing uncalculated multipliers or estimated payouts before the pool result is finalized by the admin. Show exact Bangkok timezone (`UTC+7`) for close counts and exact prediction creation time.
